Core
----

``J/K``
  Next/Prev pages.
``a/s``
  Best-fit whole page/Full width.
``d``
  Toggle dual-display.
``f/F``
  Follow links/Display link URL.
``mX/'X``
  Set mark/Goto mark.
``nG``
  N-th page.

Index
-----

``tab``
  Toggle index mode.
``l/L``
  Expand entry/All entries.
``h/H``
  Collapse entry/All entries.
